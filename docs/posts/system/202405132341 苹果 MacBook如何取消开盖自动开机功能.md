# 苹果 MacBook如何取消开盖自动开机功能？

## 1.取消开盖自动开机功能 
```bash
sudo nvram AutoBoot=%00
```

## 2.恢复自动开机功能
```bash
sudo nvram AutoBoot=%03
```
