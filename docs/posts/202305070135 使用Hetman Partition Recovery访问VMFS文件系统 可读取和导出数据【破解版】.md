- exFAT, VFAT, FAT 12 / 16 / 32 (memory card or USB drive);
- NTFS / ReFS (Windows);
- APFS / HFS+ (Apple MacOS);
- Ext2 / 3 / 4 / ReiserFS / XFS / UFS / ZFS / Btrfs (Linux, Unix, Solaris, FreeBSD, Open BSD, NetBSD);
